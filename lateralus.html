Lateralus Interactive


Website, Spring 2025

Lateralus Interactive is an interactive website about the song “Lateralus” by TOOL. The site consists of 7 pages, each with a set of lyrics. Each page was designed using the p5js library of JavaScript to be interactive, using the lyrical text as opposed to other elements of the page. I chose this song because of its overall message of transcending and pushing boundaries. I chose a typewriter-like font and dark imagery to represent an old-timey sophisticated feel. All images come from TOOL’s album covers or related imagery, which I edited using Photoshop.
